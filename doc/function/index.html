    public composeContainers(root: Element | Container, containers: ...Element | Container | String): Container source 

    import {composeContainers} from 'stardux/src/index.js'

    Compose a container from containers or DOM elements. If a Container or Element is given as first argument then it is treated as the root and all subsequent arguments are treated as direct descendants of the root. If the second argument is an array or an "array like" object then it is treated as direct descendants of the root and all subsequent arguments are ignored. If an array or "array like" object is passed as the first argument a new root container is created and the first argument is treated as direct descendants of the newly created root container. The root container, newly created or restored is returned.

    Params:

    NameTypeAttributeDescription
    root Element | Container
    • nullable: true
    containers ...Element | Container | String
    • nullable: true

    Return:

    Container

    Example:

    Compose containers together with new root.
      const a = createContainer();
  const b = createContainer();
  const composed = composeContainers([a, b]);
    Compose containers together with given root.
      const a = createContainer();
  const b = createContainer();
  const c = createContainer();
  const composed = composeContainers(a, [b, c]);
