    public set innerContents: String source

    Sets inner contents of DOM content. This will set the template source and update the container. If child containers exist in tree they will become orphaned. If value is null then the value becomes an empty string (''). undefined values result in the string 'undefined'.

    public get parent: Container | null source

    Getter to return parent container if available. Parent is determined with DOM traversal up the tree. A container can be considered orphaned if it doesn't have a parent DOM element.
